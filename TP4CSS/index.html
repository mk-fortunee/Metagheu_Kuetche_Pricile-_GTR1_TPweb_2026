Ce Mini - site a été crée dans le cadre du cours de TP programmation web.

Il permet de decouvrir la création de plusieurs pages HTML réliées entre elles.

Objectif du Mini - site

L'objectif est d'apprendre à structurer un petit projet web avec plusieurs fichiers.

Pages disponibles

  • La page Accueil présente le contexte général du Mini - site
  • La page Présentation contient les informations personnelles de l'etudiant.
  • La page contact contient les informations de contact de l'étudiant
